HypixelTracker

Item Database

This feature allows users to view and sort items based on various criteria such as name, type, and rarity.

Other Features

The website also allows the tracking of player stats and other data from their profile along with Mayor statistics and perks.

The tracking website was built using react for the frontend and node.js for the backend. The data is fetched from an open API provided by Hypixel and a few others, which supplies all the necessary information. The site features a responsive design, ensuring usability across various devices and is hosted using Microsoft Azure.
